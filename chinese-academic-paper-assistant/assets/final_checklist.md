# 最终检查清单

- [ ] 正文引用都能对应到文献矩阵。
- [ ] 未核验文献没有进入正式综述段落。
- [ ] 公开来源候选文献已统一标注为“待核验”，没有被写成正式引用。
- [ ] 每条正式文献都有作者、题名、期刊/来源、发表时间和来源状态。
- [ ] 不确定的作者、题名、期刊、年份、DOI、URL、摘要和引用关系没有被自动补编。
- [ ] 没有编造数据、实验、访谈、统计结果或文献结论。
- [ ] 没有断言未经证实的“谁参考了谁”。
- [ ] 创新点使用了谨慎表述，没有宣称“无人研究”或“首次提出”。
- [ ] 每个创新点都标注了最接近论文、相近之处、可推进空间、推荐方法论、所需数据和核验风险。
- [ ] 完整版文字式文献综述已经作为完整流程最后追加模块输出，并且所有已找到、已进入矩阵或候选表的文献都在综述中逐篇对应出现。
- [ ] 完整版文字式文献综述使用了“XXX学者在《xxxxx》一文中提出/指出/认为……”这类可追溯表达，没有把候选文献写成已核验文献。
- [ ] 中文表达自然、具体、克制，去除了明显 AI 模板句。
- [ ] 期刊推荐没有虚构核心身份、影响因子、录用率、审稿周期或版面费。
