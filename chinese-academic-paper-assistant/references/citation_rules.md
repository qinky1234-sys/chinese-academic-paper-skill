# Citation Rules

## Minimum Metadata

Formal review paragraphs require:

- Author(s)
- Title
- Journal/source name
- Publication year or date
- Source status or traceable link/identifier

If any required field is missing, keep the source in candidate status.

## Verification

Prefer DOI, official journal page, database record, Crossref, OpenAlex, or user-exported records. For Chinese databases, user-exported records are acceptable but should retain the database/source label.

## Citation Integrity

- Do not cite a paper for a claim absent from its title, abstract, user-provided excerpt, or verified metadata.
- Do not merge findings from multiple papers into one citation.
- Do not use unverified candidate sources in formal paragraphs.
- When uncertain, write `TODO: 核验该文献是否支持此表述`.

## Numbered References

If using numbered citations, assign numbers only after the user confirms the matrix. If the matrix changes, renumber all citations.
