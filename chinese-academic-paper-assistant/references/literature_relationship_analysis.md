# Literature Relationship Analysis

## Commonality Analysis

Summarize:

- Common research topic
- Common research object
- Common theory or framework
- Common method and data source
- Repeated findings or emerging consensus
- Disagreements or contradictory findings

## Timeline Analysis

Build a timeline from oldest to newest:

- Early foundation: first or earlier papers in the corpus that define the problem, theory, or method.
- Expansion: papers that extend variables, objects, cases, regions, data, or methods.
- Recent trend: papers from the latest 3-5 years that show new contexts, technologies, policies, or methods.

## Relationship Types

Use these labels:

- `理论承接`: later work uses or adapts an earlier theory.
- `方法扩展`: later work adds data, models, or empirical tests.
- `对象扩展`: later work changes sample, region, population, or industry.
- `场景迁移`: later work applies a question to a new setting.
- `观点修正`: later work challenges or qualifies an earlier conclusion.
- `并列互补`: papers address adjacent parts of a topic.
- `直接引用`: only when citation data confirms it.

## Caution Rules

Do not claim direct reference/citation without citation metadata. If only chronology and topic similarity are available, write:

> 从发表时间和研究内容看，后续研究可能承接了前文的问题意识或分析框架，但是否存在直接引用关系仍需进一步核验。

When source abstracts are missing, mark relationship claims as tentative.
