# 复制即用提示词模板

把下面模板中的【】内容替换成自己的主题、方向或文献资料即可使用。建议每次只做一个明确任务，先整理文献，再写综述。

## 1. 第一次使用：导入文献并生成矩阵

```text
使用 chinese-academic-paper-assistant。

我的论文主题是【填写论文主题】。
研究方向是【填写研究方向】。
论文类型是【本科论文/硕士论文/期刊论文】。

下面是我已经找到的文献资料，可能来自 CNKI、万方、维普、Zotero、BibTeX、RIS 或 PDF 摘要。
请你先识别资料格式，再整理成文献矩阵。

要求：
1. 检查作者、题名、期刊/来源、发表时间、摘要、关键词、DOI/链接是否完整。
2. 字段不完整的文献放入“待核验文献”，不要写进正式综述。
3. 不要编造文献、结论、期刊信息或发表时间。
4. 最后告诉我还需要补充哪些字段。

【在这里粘贴文献资料】
```

## 4. 从文献矩阵生成文献综述

```text
使用 chinese-academic-paper-assistant。

下面是我已经整理好的文献矩阵。
请基于这些文献生成一段可以放入论文【引言/文献综述/开题报告】中的中文综述。

要求：
1. 先概括文献共同点。
2. 再说明研究脉络：谁较早提出，谁可能延伸或回应了前人的研究。
3. 只有在引用数据明确时，才说“谁引用了谁”。
4. 没有证据时，用“从发表时间和研究内容看，可能存在承接关系，需进一步核验”。
5. 文字要自然，减少 AI 味，不要使用空泛套话。
6. 保留作者和年份引用标记。
7. 不要添加矩阵中没有的文献或结论。

【在这里粘贴文献矩阵】
```

## 10. 完整工作流

```text
使用 chinese-academic-paper-assistant。

我的论文主题是【填写主题】。
研究方向是【填写方向】。
论文类型是【本科论文/硕士论文/期刊论文】。
目标引用风格是【GB/T 7714/APA/学校格式/暂不确定】。

请按完整流程处理：
1. 识别并整理我提供的文献资料。
2. 如果我没有提供文献，请先生成中文数据库检索策略，并从公开来源查找候选文献。
3. 生成首批文献主张表：作者+年份+一句话核心主张。
4. 按共同假设分组，标记互相矛盾或存在张力的论文。
5. 绘制宏观背景图。
6. 生成文献矩阵。
7. 检查字段缺失和待核验文献。
8. 分析文献共同点。
9. 分析研究脉络和逻辑关系。
10. 生成研究领域知识地图。
11. 总结研究空白。
12. 给出 3-5 个可选创新点，每个创新点都要附最接近论文、相近之处、可推进空间、推荐方法论和所需数据。
13. 根据论文类型说明本科/硕士/期刊论文输出重点。
14. 生成一段去 AI 化中文文献综述。
15. 如需要期刊方向，只给出期刊匹配思路和人工核验清单。
16. 告诉我下一步应该补充哪些文献或材料。
17. 最后追加完整版文字式文献综述，不改变前面任何输出；所有已经找到并进入矩阵或候选表的文献都必须逐篇对应写到，格式为“XXX学者在《xxxxx》一文中提出/指出/认为……”。

边界：
不要编造参考文献。
不要把待核验文献写进正式综述。
不要绕过数据库登录、付费、验证码或学校权限。

【在这里粘贴文献资料】
```
