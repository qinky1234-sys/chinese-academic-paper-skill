# 3 分钟上手

这份文档只解决一个问题：第一次使用 `chinese-academic-paper-assistant` 时，应该从哪里开始。

## 第 1 步：先判断你有没有文献

选择一种情况：

| 情况 | 你应该怎么做 |
| --- | --- |
| 已经有文献 | 先粘贴 CNKI、万方、维普、学校图书馆、期刊官网、Zotero、BibTeX、RIS 或 PDF 摘要资料，让 skill 先生成首批文献主张图，再生成文献矩阵 |
| 还没有文献 | 先输入论文主题、研究方向和论文类型，让 skill 查找公开来源候选文献，并生成待核验版首批文献主张图和中文数据库检索策略 |
| 只有几篇文献 | 先导入已有文献，再让 skill 判断是否需要补充候选文献，并把已有文献和候选文献分开标注 |

## 第 2 步：选择论文模式

直接告诉 skill 你要哪种模式：

```text
请按本科论文模式输出。
请按硕士论文模式输出。
请按期刊论文模式输出。
```

如果你不确定，按这个选：

| 你的任务 | 推荐模式 |
| --- | --- |
| 课程论文、本科毕业论文 | 本科论文模式 |
| 硕士论文、开题报告、研究设计 | 硕士论文模式 |
| 投稿论文、期刊论文引言或文献综述 | 期刊论文模式 |

## 第 3 步：复制对应提示词

### 情况 A：我已经有文献

```text
使用 chinese-academic-paper-assistant。

我的论文主题是【填写主题】。
研究方向是【填写方向】。
论文类型是【本科论文/硕士论文/期刊论文】。

下面是我已经找到的文献资料，可能来自 CNKI、万方、维普、学校图书馆、期刊官网、Zotero、BibTeX、RIS 或 PDF 摘要。

请先完成：
1. 识别资料格式。
2. 输出每篇文章的“作者+年份+一句话核心主张”表。
3. 按共同假设进行分组。
4. 标记互相矛盾或存在张力的论文。
5. 绘制宏观背景图，而不是普通总结。
6. 整理成文献矩阵。
7. 检查作者、题名、期刊/来源、发表时间、摘要、关键词、DOI/链接是否完整。
8. 标出可进入正式综述的文献和需要人工核验的文献。
9. 分析文献共同点、研究脉络、研究空白和可选创新点。

要求：
- 不要编造文献。
- 不要把字段不完整的文献写进正式综述。
- 一句话核心主张必须来自题名、摘要、关键词、结论或我提供的摘录。
- 如果没有引用证据，不要直接说“谁引用了谁”。

【在这里粘贴文献资料】
```

### 情况 B：我还没有文献

```text
使用 chinese-academic-paper-assistant。

我的论文主题是【填写主题】。
研究方向是【填写方向】。
论文类型是【本科论文/硕士论文/期刊论文】。
我现在还没有整理好的文献。

请先完成：
1. 拆解核心关键词、同义词和扩展词。
2. 生成 CNKI、万方、维普、学校图书馆和期刊官网可用的检索策略。
3. 从公开来源查找候选文献。
4. 如果候选文献元数据足够，生成待核验版“作者+年份+一句话核心主张”表。
5. 按共同假设对候选文献分组，标记可能存在矛盾或张力的论文。
6. 绘制待核验版宏观背景图。
7. 生成待核验版文献矩阵。
8. 基于待核验文献生成初步文献综述、研究脉络、研究空白和创新点。

要求：
- 候选文献必须标注为“待核验”。
- 待核验版主张表和宏观背景图只能作为检索和判断方向的草稿。
- 待核验文献不要当成正式可引用文献。
- 不要绕过 CNKI、万方、维普、学校图书馆的登录、付费、验证码或权限限制。
```

### 情况 C：我只有几篇文献，需要补充

```text
使用 chinese-academic-paper-assistant。

我的论文主题是【填写主题】。
研究方向是【填写方向】。
论文类型是【本科论文/硕士论文/期刊论文】。
我现在已经有【填写数量】篇文献，但可能不够。

请先整理我已有的文献，再判断是否需要补充检索。

请输出：
1. 已有文献的作者+年份+一句话核心主张表。
2. 共同假设分组。
3. 互相矛盾或存在张力的论文。
4. 宏观背景图。
5. 已有文献矩阵。
6. 当前文献覆盖了哪些主题。
7. 还缺哪些类型的文献。
8. 需要补充的关键词和检索式。
9. 可以从公开来源补充的候选文献。
10. 待核验版研究空白和创新点建议。

【在这里粘贴已有文献资料】
```

## 第 4 步：看懂输出里的标记

输出中常见的标记含义：

| 标记 | 含义 | 能否直接写进正式论文 |
| --- | --- | --- |
| 可进入正式综述 | 作者、题名、来源、年份和标识信息较完整 | 可以，但仍建议人工复核 |
| 待核验文献 | 字段不完整或来源尚未确认 | 不建议直接写入 |
| 补充候选文献 | 来自公开来源或补充检索，等待用户确认 | 不建议直接写入 |
| 首批文献主张图 | 作者+年份+一句话核心主张、共同假设、矛盾/张力和宏观背景 | 可用于理解领域，正式使用前要复核来源 |
| 待核验版综述 | 基于候选文献生成的初步段落 | 只能作为草稿 |
| 需进一步核验 | 引用关系、期刊信息或结论证据不足 | 核验后再使用 |

## 第 5 步：继续做下一步

拿到第一轮输出后，建议这样继续：

1. 先补齐缺失字段。
2. 删除明显不相关的候选文献。
3. 检查一句话核心主张是否准确。
4. 确认共同假设分组和矛盾标记是否合理。
5. 确认哪些文献可以进入正式综述。
6. 再让 skill 写去 AI 化中文综述段落。
7. 最后检查引用、期刊信息和研究结论。

可以继续这样问：

```text
请基于已经确认的文献，生成一段可以放入【引言/文献综述/开题报告】的中文综述。
要求保留作者和年份，说明文献共同点、研究脉络和研究空白。
文字要自然，减少模板句，不要加入未核验文献。
```
