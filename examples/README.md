# Examples

这些示例展示 `chinese-academic-paper-assistant` 的典型输出形态，方便用户在安装前快速判断它是否适合自己的中文论文写作流程。

注意：示例中的文献题名、作者和期刊信息用于演示格式，不应当直接作为真实参考文献使用。正式写作时，请使用用户从 CNKI、万方、维普、学校图书馆、期刊官网、DOI、BibTeX、RIS、EndNote、NoteExpress 或 PDF 中导入的真实文献。

## 推荐演示路径

如果你想向别人展示这个 skill，建议按这个顺序：

1. 先打开示例 1，让用户看到“题录如何变成矩阵”。
2. 如果用户不知道怎么导出文献，打开示例 4，让用户照着 Zotero/BibTeX/RIS 路径操作。
3. 再打开示例 2，让用户看到“文献不够时如何补充检索”。
4. 打开示例 8，让用户看到“没有文献也能开始”的全能力端到端效果。
5. 打开示例 6，让用户理解本科、硕士、期刊论文三种输出深度的区别。
6. 打开示例 7，让用户看到“创新点如何对应最接近论文和方法论”。
7. 打开示例 5，查看更短的完整中文论文场景案例。
8. 打开示例 9，让用户看到“最后如何追加逐篇覆盖所有文献的文字版综述”。
9. 最后打开示例 3，让用户看到“矩阵如何变成论文里能用的自然综述段落”。
